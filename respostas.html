  1. Internet é a rede de conexões, ja a Web é um dos vários meios para utilizar os conteúdos da internet.
  2. Clientes são os usuários da Web, conectados na Internet, e em programas de acesso à navegação como os browser (chrome, firefox, safari etc.). Já os servidores são computadores que armazenam dados.

  4. O HTML, independente da versão em que esteja, não é uma linguagem de programação, visto que, como o proprio nome diz, é uma linguagem de marcação (markup language).
  1. Listas de definição servem para marcar termos e definições. As tags usadas são <dl>, <dt> e <dd>. Onde <dl> (definition list) cria a lista de definição, <dt> (difinition term) onde o termo é proposto e <dd> (definition definition) onde o termo é definido.
